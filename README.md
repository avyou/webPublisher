## webPublisher
------------
#### 介绍

这是一个用于网站的web文件发布工具，与SVN结合方便的将web文件更新或发布到正式网站上。同时可以实现网站文件更新时自动备份与还原，包括SVN提交，SVN更新，文件更新与备份列表的日志查看等功能。

#### 项目地址
https://github.com/avyou/webPublisher

#### 使用框架
wxpython
